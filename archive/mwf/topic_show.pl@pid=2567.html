The 17th Ludum Dare compo again had some interesting entries. While I think none won a pelly, each game has some interesting technical aspects that I really like, so I am adding them again. :P

Rising Sea Levels by jsb, a shader-powered sinking-island, rescue-people game in Ruby
Capsize! by philomory, a Ruby balancing game
ChickenPost! by pythong, a 3D cartoon obstacle run in Ruby
SautEUR by ickylevel, a fast Ruby 2D platformer playing as a fish :D
Space Island by mathias, an unfinished C++ RTS with a nice UI and polygonal resources
