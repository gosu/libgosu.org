Have you tried defining #button_down(id) and #button_up(id) instead of using the Window#button_down?(id) method? The two should be equivalent, but maybe there's a bug or something?

example:
class Window < Gosu::Window
  def button_down(id)
    if id == Gosu::KbUp
      @player.up
    end
  end
end
