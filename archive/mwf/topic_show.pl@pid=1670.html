What are you trying to do? For text input (e.g. highscore list names), the TextInput class should be used as shown in the TextInput.rb example.

For other use well, there currently is a gap because the previous button_id_to_char/char_to_button_id methods have been silently deprecated and don't work on OS X 10.6. :|

By jlnr (dev) Date 2009-11-03 16:09
Oh, that sounds like a project you should post to the Showcase when it works. :)

Unless on OS X 10.6, button_id_to_char still works, so you can check button_id_to_char(id) in the Window#button_down callback. It is safer to use a TextInput object though. You would just create one and set Window#text_input= to that instance, then check for the last character of its string (resetting it to the empty string every frame if desired).
