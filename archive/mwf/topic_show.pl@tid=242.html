I never made a proper post for my Ludum Dare #15 Gosu entry, so here goes. (so I have a chance at project of the day ;)..).

== INSTALL
1) gem install chingu
2) Download source from http://github.com/ippa/the_eternally_grey
3) Run "the_eternally_grey.rb"

Works best with 1.9.1+ but should also work with 1.8.7+

== PLAY
The Eternally Grey, solid and still for thousand of years. Suddenly you feel something you haven't felt in ages...

== CONTROLS
Mouse / Left click. You'll figure out where.

== DEVELOPMENT
Built with Ruby 1.9.1, http://www.libgosu.org and http://github.com/ippa/chingu/tree/master

Music and SFX with http://reaper.fm/ + VSTs

Win32 build with http://github.com/larsch/ocra/tree/master
