Topic Gosu / Extending Gosu / Recent Precompiled OSX Texplay Package?
By RavensKrag Date 2011-12-30 04:52
Can someone make a precompiled gem for OSX of texplay? I don't own a mac, and don't want to force the artists I'm working with to install xcode just to install this library.

I would greatly appreciate it.

Side question: would a precompiled package for use with C Ruby still work with MacRuby?
By jlnr (dev) Date 2011-12-30 04:55
MacRuby: No.

Texplay comes precompiled inside the wrapper .app bundle, in the download folder linked from the www.libgosu.org front page. It should be in RubyGosu.app/Contents/Resources/lib/texplay*.

Alternatively, you could just move your source in there and hand the whole .app over to your artist.
By RavensKrag Date 2011-12-30 06:38
Oh, thanks.

I somehow imagined the process of packing of the .app to be more complicated.  I didn't realize that an .app was just a fancy directory.
By jlnr (dev) Date 2011-12-30 11:08
Don't worry, it gets exponentially more complicated if you need third-party binary gems and want to run on all architectures / OS X versions :)
