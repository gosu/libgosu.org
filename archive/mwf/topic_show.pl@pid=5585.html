By jlnr (dev) Date 2012-02-24 06:57
You cannot draw things inside button_down(id). You have to remember somewhere that you want a new tile to be there, and then draw if every frame.

You can see how CptnRuby.rb implements a tile-based map and draws all the tiles in the array every frame. If the user clicked at a tile, you would then store an index or image reference in the tiles array. It makes sense to share the Map class between a game and its editor...to get started, you can implement a method to save Cptn Ruby's Map class to a file, and then add level editing right into the game.

Also, you can make your conditional shorter if you use:
elsif mouse_x.between?(0, 39) and mouse_y.between?(0, 39)
or this (parentheses only where absolutely necessary):
elsif (0..39).include?(mouse_x) and (0..39).include?(mouse_y)
