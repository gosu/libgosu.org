Gosu now uses AVAudioPlayer for Gosu::Song playback if the file ends in .mp3, .m4a or .aac. This makes playback way faster because it is hardware accelerated (.ogg is never).

For sound, I heard IMA4 is the way to go. Will inspect that too.

Funny how the distinction between Song and Sample makes sense on so many platforms. I guess the merger into one universal Gosu::Sound object is never going to happen.
