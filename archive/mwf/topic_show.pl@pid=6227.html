By jlnr (dev) Date 2012-09-17 06:38
I am in the progress of updating everything to use Xcode 4. If you want to do it the "new" way, you should start a new C++ project and drop the ios/GosuTouch.xcodeproj into your project. Then you should add "Gosu" (not "Gosu Touch") as a dependency for your game's target. It's basically the same process now as on iOS, which is described here (other library, same idea):

http://code.google.com/p/core-plot/wiki/UsingCorePlotInApplications

If a file is red, it generally means it doesn't exist. That's perfectly normal if you open Gosu.xcodeproj - because you haven't built it.
