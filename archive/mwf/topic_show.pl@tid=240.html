By jlnr (dev) Date 2009-10-02 08:02
Gosu does not provide any framework for any genre of games, so it should be just as good or bad for any kind of game.

If you have a class to hold the map, adding animation should be very trivial. Gosu does not support .gif files if you want to literally use some as your base gfx, but maybe you can come up with a short RMagick script to dissect them into tiled PNG files? I have no experience with this ;)
