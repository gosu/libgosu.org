By Spooner Date 2012-01-09 16:20
The problem is that the path to the image is almost certainly not correct. I think you might mean to be using this:

@iWater = Image.new( self, "media/sprite/chipset/main.png", true )

Also it is not a good idea to load the image once _every_ frame. Best to load the image in #initialize, rather than #draw.

EDIT: Your error refers to an error in #load_tiles, but your example just loads a single image. Assume that you have the same error in both cases though.
