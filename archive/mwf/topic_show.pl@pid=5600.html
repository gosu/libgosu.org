By Spooner Date 2012-02-27 14:04
Don't use #paint in Ruby 1.9.3:

@image.rect 0, 0, @image.width, @image.height, :color => [0, 0, 0, 0], :fill => true

@image.pixel rand(@image.width), rand(@image.height), color => :red

#paint was removed in TexPlay 0.4.x, since it isn't compatible with Ruby 1.9.3.

#paint is completely unnecessary in this code anyway!

Edit: You can write the first line (#rect):

@image.clear
