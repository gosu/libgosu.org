By jlnr (dev) Date 2014-12-06 20:39
Just another maintenance release:

* Windows: Add support for Ruby 2.1.x
* All: Check SDL 2 return values when creating the Window (helpful when there is no OpenGL capable video driver)

Also, the release process is a little messed up. There have been no 0.8.x source tarballs so far. The new directory structure in 0.8.0 also broke the CMake setup. I'll try to fix these issues and follow up with a working 0.8.6 source tarball. Hopefully that will make it easier to get C++ Gosu into package managers such as Homebrew (see http://www.libgosu.org/cgi-bin/mwf/topic_show.pl?tid=1160).
