c:/ruby/bin/ruby.exe extconf.rb
The Gosu gem requieres some libraries to be installed system-wide.

and then:

*** extconf.rb failed ***

Any solution?
By banister Date 2010-07-17 04:58
try:

gem install gosu --platform=mswin32
By Spooner Date 2010-07-17 08:32
However, if you used the Ruby Installer, then you'd need to use:

  gem install gosu --platform=mingw32

The current Ruby Installer uses ruby/gems compiled with mingw, but the older version used VC6 (mswin32). The latest version of gosu is only compiled in mingw, so it is best to use the proper Ruby Installer version anyway or you will only have access to Gosu 0.7.18.
By jlnr (dev) Date 2010-07-17 08:55 Edited 2010-07-17 08:59
What you have to pass depends on Gosu, not on the installed Ruby version. Since the mingw32-based Ruby Installers became popular, Gosu's platform string for the Windows version is mingw32.

The most accurate string right now is: gem install gosu --platform=i386-mingw32
